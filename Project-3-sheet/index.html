In 2023, my team and I participated in a hackathon hosted by the IT company Modsen. The theme of the hackathon was “Smart City.” After much deliberation, we settled on the idea that a truly smart city should prioritize cleanliness. Our solution? A chatbot named “Smart Trash.” This chatbot would assist users in locating the nearest trash cans, guide them on proper waste disposal, and even share interesting recycling facts to raise awareness. By combining technology with environmental consciousness, we aimed to contribute to a cleaner, more informed urban landscape.

When embarking on any project, having a solid technology stack is crucial. For our “Smart Trash” chatbot, we carefully selected the following technologies:
Python Flask: A lightweight web framework that allows us to build the backend of our chatbot efficiently. Flask`s simplicity and flexibility make it an excellent choice.
JavaScript: Essential for creating dynamic and interactive user interfaces. We used it to enhance the chatbot`s frontend experience.
HTML and CSS: These form the backbone of web development. HTML structures our content, while CSS styles it beautifully. Together, they ensure a polished user interface.
SQL: Our database management system. SQL will help us store information about trash cans, recycling centers, and user interactions.
Additionally, we integrated an image recognition API to identify objects in photos. Unfortunately, due to limitations at the time of implementation, we couldn`t utilize the ChatGPT API directly. Instead, we manually crafted responses to ensure the chatbot remained functional. Rest assured, once the ChatGPT API becomes available, we can enhance the project further and achieve our original vision.
